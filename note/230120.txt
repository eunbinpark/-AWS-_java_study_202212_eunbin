	DML
    C : insert / into	데이터추가
    R : select / from	데이터 조회
    U : update / set	데이터 수정
    D : delete / from	데이터 삭제

데이터테이블에 값 넣는 법

insert into 
데이터테이블.코드(벨류이름1, 벨류이름2, 벨류이름3)
values 
(벨류1, 벨류2, 벨류3)

각각의 벨류에 맞는 값을 넣어야함(순서는 상관없음)
만약 값을 없는채로 두고 싶다면 벨류 설정을 널 허용으로 해줘야 함
그냥 데이터테이블에 값을 추가하고 싶으면 컬럼을 안넣어도 되는데 그렇게 하면 널을 넣고싶을때 모든 널인 곳에 다 각자 값을 널로 넣어줘야 함
그래서 컬럼이름을 따로 지정해주고 해당값도 따로 지정해줌으로써 필요한 값만 넣을수 있게 됨

데이터테이블 값 수정하는 법

update 
데이터테이블
set
바꿀 벨류이름 = 바꿀 벨류
where
바꾸고 싶은 위치(보통 키값)
값을 한번에 여러개 바꾸게 되면 위험하기 때문에 에디터가 막아버림
해결방법 : 에디트 - 프리퍼런스 - sql에디터 - 세이프 해제 하고 껐다 켬

데이터테이블 값 삭제하는 법

delete from 데이터테이블
where
삭제할 벨류
(추가하고 싶으면 and로 연결)
